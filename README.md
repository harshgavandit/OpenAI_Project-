## 💻 Tech Stack

### Frontend
- **Framework**: Next.js 16 (React 19)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Visualization**: D3.js
- **State**: React Hooks
- **HTTP**: Fetch API

### Backend
- **Framework**: FastAPI
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy
- **Auth**: JWT
- **AI**: OpenAI GPT

### Hosting
- **Frontend**: Vercel
- **Backend**: Render
- **Database**: Neon PostgreSQL
